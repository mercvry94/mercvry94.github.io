<html>

HTML tags Function
<head> contains the invisible parts of the website, the title, link to stylesheets
<body> contains the content of the website, the images and texts
<nav> unordered list at top of site
<img> self-closing, image tag src = ""
<div> used to seperate paragraphs and navigate easier, with nav bar and attributes
<li> list item, <ul> or <ol>
<table> creates a table, <th> for headers <tr> for the rows and <td> for the cells themselves
